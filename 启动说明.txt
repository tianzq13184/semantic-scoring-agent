⚠️ 重要：启动后端 API 的正确方式

❌ 错误的方式（会导致导入错误）：
   cd api
   uvicorn main:app --reload --port 8000

✅ 正确的方式：
   1. 确保在项目根目录（semantic-scoring-agent）
   2. 运行：uvicorn api.main:app --reload --port 8000

完整步骤：

终端 1 - 启动后端：
   cd /Users/kurttian/Desktop/semantic-scoring-agent
   uvicorn api.main:app --reload --port 8000

终端 2 - 启动前端：
   cd /Users/kurttian/Desktop/semantic-scoring-agent/ui
   streamlit run app.py

注意：使用 api.main:app 而不是 main:app
